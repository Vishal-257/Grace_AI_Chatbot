# React + TypeScript + Vite

This is Grace AI, chatbot built using React+TypeScript+Vite. It is a fully responsive website which lets you save your session locally, rename chat session or delete it. You can also export your chat data in JSON format.

Installation Guide:
1. Create a new folder.
2. Download the project files in your folder: git clone repo-url
3. In your terminal cd to the respective folder: cd project-folder
4. Install all the dependencies: npm install
5. Get your Gemini API from https://aistudio.google.com/ and paste it in .env file by name VITE_API_KEY: "your api key"
6. Now run the project by: npm run dev
The project will be live on http://localhost:5173/
To run the project in Production Mode:
npm run build
npm run preview
